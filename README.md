# MyNote (iOSアプリ勉強用)
----

# ビルド

    $ bundle install
    $ bundle exec pod install
    $ // double click MyNote.xcworkspace

# 機能 
- リスト表示
- 詳細(作成、編集)
- 文字サイズ変更
